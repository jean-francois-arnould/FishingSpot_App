# Assets pour AltStore

Ce dossier contient les ressources nécessaires pour la distribution AltStore.

## Fichiers requis

### 1. `icon.png` (512x512 px)
Icône principale de la source AltStore

### 2. `fishingspot-icon.png` (512x512 px)
Icône de l'application FishingSpot

### 3. `header.png` (1200x600 px)
Image d'en-tête pour la source AltStore

### 4. Captures d'écran (1170x2532 px pour iPhone 12 Pro)
- `screenshot1.png` - Page d'accueil avec liste des captures
- `screenshot2.png` - Détails d'une capture
- `screenshot3.png` - Page de statistiques avec graphiques

## Générateur d'icônes

Vous pouvez utiliser ces outils pour créer vos icônes :

- **Figma** (gratuit) : https://www.figma.com
- **Canva** (gratuit) : https://www.canva.com
- **AppIcon.co** : https://appicon.co

## Templates recommandés

### Icône (512x512 px)
- Fond : Dégradé bleu (#4A90E2 → #2E5A8B)
- Logo : 🎣 ou illustration de poisson
- Texte : "FishingSpot" (optionnel)

### Header (1200x600 px)
- Image de fond : Paysage de pêche (lac, rivière)
- Overlay semi-transparent
- Titre : "FishingSpot"
- Tagline : "Votre carnet de pêche numérique"

### Screenshots
Utilisez l'émulateur iOS dans Visual Studio pour capturer :
1. CMD + S dans le simulateur
2. Les captures sont sauvegardées sur le bureau
