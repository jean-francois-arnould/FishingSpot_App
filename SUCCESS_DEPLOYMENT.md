## 💡 OPTION : URL Personnalisée (Domaine Custom)

Si vous voulez une URL plus courte comme `fishingspot.com` ou `app.fishingspot.com` :

1. **Acheter un domaine** (chez Namecheap, GoDaddy, etc.)
2. **Configurer le DNS** :
   - Type : `CNAME`
   - Host : `@` ou `www`
   - Value : `jean-francois-arnould.github.io`
3. **Configurer dans GitHub** :
   - Settings → Pages → Custom domain
   - Entrer votre domaine
   - Cocher "Enforce HTTPS"

Avec un domaine personnalisé, l'URL devient :
```
https://fishingspot.com/         ← Sans le préfixe !
https://fishingspot.com/login    ← Direct !
```
