# 🎣 FishingSpot - Installation via AltStore

## Installation

### Prérequis
- iPhone/iPad avec iOS 15.0 ou supérieur
- [AltStore](https://altstore.io/) installé sur votre appareil
- AltServer en cours d'exécution sur votre ordinateur

### Étapes d'installation

1. **Ouvrez AltStore** sur votre iPhone/iPad

2. **Ajoutez la source FishingSpot** :
   - Appuyez sur l'onglet "Sources"
   - Appuyez sur le bouton "+" en haut à droite
   - Entrez l'URL : `https://jean-francois-arnould.github.io/FishingSpot_App/altstore-source.json`
   - Appuyez sur "Add"

3. **Installez l'application** :
   - Trouvez "FishingSpot" dans la liste des applications
   - Appuyez sur "INSTALL"
   - Attendez que l'installation se termine

4. **Premier lancement** :
   - Allez dans Réglages > Général > Gestion des profils
   - Faites confiance au développeur
   - Lancez FishingSpot depuis votre écran d'accueil

## Mise à jour

AltStore vérifie automatiquement les mises à jour. Pour mettre à jour manuellement :
1. Ouvrez AltStore
2. Allez dans l'onglet "Mes Apps"
3. Appuyez sur "Rafraîchir tout"

## Fonctionnalités

✅ **Enregistrement des captures**
- Photos avec géolocalisation GPS
- Informations détaillées (taille, poids, lieu)
- Notes personnalisées

✅ **Météo en temps réel**
- Conditions météo au moment de la capture
- Historique des conditions de pêche

✅ **Statistiques**
- Graphiques de vos performances
- Analyse par poisson, lieu, période
- Records personnels

✅ **Export de données**
- Export PDF pour impression
- Export CSV pour analyse
- Partage sur réseaux sociaux

✅ **Calendrier de pêche**
- Vue mensuelle de vos sorties
- Statistiques par jour

✅ **Gestion de matériel**
- Catalogue de votre équipement
- Setups de pêche personnalisés

✅ **Mode hors ligne**
- Toutes les fonctionnalités disponibles sans connexion
- Synchronisation automatique
